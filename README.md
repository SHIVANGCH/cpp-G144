                                                                                HOTEL MANAGEMENT SYSTEM :

                                                                                
A robust, console-based application designed to streamline hotel operations, including room booking, guest record management, and automated billing. This project demonstrates core Object-Oriented Programming (OOP) principles and File Handling in C++.



                                                                                 KEY FEATURES :
1) Guest Check-In/Check-Out: Seamlessly register guests and assign room numbers.

2) Room Allotment: Logic to prevent double-booking of rooms.

3) Automated Billing: Instant invoice generation based on room type and duration of stay.

4) Persistent Storage: Data is stored in a binary file (hotel_records.dat), ensuring information is saved even after the program closes.

5) Search Functionality: Quickly find guest details using a Room Number or Name.

                                                                                 TECHNICAL IMPLEMENTATION :

This project was built to demonstrate proficiency in:

1. Object-Oriented Programming (OOP)
Encapsulation: Using private data members and public methods to protect data integrity.

Inheritance: Implementing a base Person class and a derived Customer class to manage shared attributes.

Polymorphism: Overloading functions for flexible data display.



2. File Handling
Utilizes the fstream library for CRUD (Create, Read, Update, Delete) operations.

Binary file modes (ios::binary) are used for efficient data storage and retrieval.



3. User Interface
Menu-Driven: A user-friendly switch-case menu for easy navigation.

Formatted Output: Uses the <iomanip> library to present guest data in clean, readable tables.
